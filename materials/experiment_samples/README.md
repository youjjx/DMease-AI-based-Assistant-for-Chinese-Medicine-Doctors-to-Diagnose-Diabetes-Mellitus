# Experiment Sample Materials

This folder contains reproducible experiment materials generated from the configured DMease pipeline.

## Tables

- `experiment_summary_P2025-0001.csv`: high-level inference summary.
- `inference_result_P2025-0001.json`: complete structured inference output.
- `ranked_candidates_P2025-0001.csv`: ranked herb candidates and target coverage.
- `prescription_steps_P2025-0001.csv`: sequential prescription steps and residual symptom state.
- `symptom_target_scores_P2025-0001.csv`: symptom-target association scores.
- `configured_kg_triples.csv`: configured knowledge graph triples used by the sample pipeline.

- `figures/herb_ranking_P2025-0001.png`: ranked herb recommendation scores.

- `figures/symptom_target_heatmap_P2025-0001.png`: symptom-target association heatmap.
